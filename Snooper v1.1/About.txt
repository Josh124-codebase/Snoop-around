 ABOUT SNOOPER
================

--*-- Overview --*--

Snooper is a lightweight command-line file system search utility written in Python.

Its goal is to provide a fast and straightforward way to search through directories, locate files or file extensions, and inspect directory structures directly from the terminal. Snooper emphasizes simplicity, portability, and a familiar command-line experience.

Version 1.1 introduces recursive searching, extension-based searches, filename searches, and the ability to display a directory tree.

---

--*-- Features --*--

* Recursive filesystem traversal.
* Search by filename.
* Search by file extension.
* Exact or partial filename matching.
* Optional case-insensitive searching.
* Configurable search root directory.
* Directory tree display.
* Detection and marking of hidden files and folders.
* Graceful handling of inaccessible directories and filesystem errors.
* Cross-platform design using Python's standard library.

---

--*-- Command Line Options --*--

 -t   ->   Search for one or more filenames.

 -e  ->   Search for one or more file extensions.

 -s   ->   Perform an exact match instead of a partial match.

 -i   ->   Ignore character case during comparisons.

 -D ->   Specify the root directory where the search begins.

 --tree  ->  Display the directory tree beginning at the specified root directory.

 -V, --version ->  Display the current Snooper version.

examples : 
	-> snooper.py -s  -i -t example.txt example2.txt etc.txt
	->snooper.py -e .txt .png .csv -D C:\
	-> snooper.py -V
	-> snooper.py --version
	             ---------------------------------------------------------------------------------
	             |     NOTE: -e and -t flags cannot be used together, this raises an error.      |
	             ---------------------------------------------------------------------------------
---

--*-- How Snooper Works --*--

Snooper recursively traverses the directory tree using Python's `os.scandir()` interface. During traversal it compares filenames or extensions against the search targets provided by the user.

When displaying a directory tree, Snooper:

* Lists directories before files.
* Uses ASCII tree connectors for readability.
* Marks hidden files and directories with `[H]`.
* Continues execution when permission or filesystem errors are encountered whenever possible.

---

--*-- Design Goals --*--

Snooper is built around a few core principles:

* Fast directory traversal.
* Minimal dependencies.
* Clean command-line interface.
* Cross-platform compatibility.
* Easy to extend with future functionality.

The project intentionally relies on Python's standard library wherever practical, making Snooper lightweight and easy to run on any system with Python installed.
